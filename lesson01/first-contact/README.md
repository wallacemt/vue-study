# Lição 01 — First Contact

Primeiro contato com Vue 3: estrutura de um projeto criado com `create-vue`, sintaxe do `<script setup>` e composição básica de componentes.

## Tópicos estudados
- Estrutura de um projeto Vue 3 + Vite + TypeScript
- `<script setup lang="ts">`
- Criação e importação de um componente filho (`Button.vue`)
- `ref()` e manipulação de eventos (`@click`)

## Rodando o projeto
```sh
bun install
bun run dev
```
