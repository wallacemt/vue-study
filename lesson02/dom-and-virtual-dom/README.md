# Lição 02 — DOM x Virtual DOM

Exploração de como o Vue reconcilia mudanças de estado com o DOM real através do Virtual DOM, sem manipulação manual do DOM.

## Tópicos estudados
- Diferença entre DOM e Virtual DOM
- Re-renderização reativa a partir de mudanças de estado (`ref()`)
- Renderização condicional com `v-if`
- `watchEffect()` para observar efeitos colaterais e visualizar o ciclo de reatividade no console

## Rodando o projeto
```sh
bun install
bun run dev
```
