# Lição 03 — Reactivity in Vue

Comparação prática entre `ref()` e `reactive()`, os dois principais primitivos de reatividade da Composition API.

## Tópicos estudados
- `ref()` para valores primitivos (contador com incremento/decremento/reset)
- `reactive()` para objetos (formulário controlado com `v-model`)
- Unwrapping automático de `ref` dentro do template
- Quando usar `ref` vs `reactive`
